Enable Real-Time Device Sync

To magically sync this app across your phone and computer, you just need a free Google Firebase link.

  1. Go to console.firebase.google.com and sign in.
  2. Click Add Project (Name it 'PersonalCFO', turn off Analytics).
  3. Click the Web icon (</>) to add an app.
  4. Copy ONLY the JSON text inside the firebaseConfig = { ... } box.
  5. Go to Firestore Database (left menu) -> Create Database -> Start in Test Mode.
